Description

Descripción: Esta tabla contiene los detalles sobre los productos de maquillaje disponibles en la tienda. Incluye información sobre la fecha de expiración, tipo de maquillaje, tono, y la cantidad de stock disponible.

idmaquillaje (INT): Clave primaria, identificador único y autoincremental para cada producto de maquillaje. fechaexpiracion (DATE): Fecha en la que el maquillaje expira. idtipomaquillaje (INT): Llave foránea que referencia a la tabla tiposmaquillaje. Define el tipo de maquillaje, como base, labial, sombra de ojos, etc. idtono (INT): Llave foránea que referencia a la tabla tonos. Especifica el tono o color del maquillaje. idproducto (INT): Llave foránea que referencia a la tabla productos. El maquillaje está vinculado a la tabla de productos generales. stock (INT): Cantidad de unidades de maquillaje disponibles en inventario. Relaciones:

idtipomaquillaje referencia a tiposmaquillaje(idtipomaquillaje): Relaciona el producto de maquillaje con un tipo específico de maquillaje. idtono referencia a tonos(idtono): Vincula el maquillaje con un tono o color específico. idproducto referencia a productos(id_producto): Cada producto de maquillaje también está registrado en la tabla de productos generales.
