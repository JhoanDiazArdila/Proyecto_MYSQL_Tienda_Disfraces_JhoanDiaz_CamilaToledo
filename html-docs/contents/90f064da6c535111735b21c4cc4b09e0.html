Proyecto_Base_datos :: Tienda_disfraces_Doña_Ruca :: transacciones

Description

Descripción: Esta tabla contiene información sobre las transacciones realizadas en la organización.

idtransacion (INT): Clave primaria, identificador único y autoincremental para cada transacción. valortotal (DECIMAL): Valor total de la transacción, representado como un valor decimal. Este campo es obligatorio y no puede ser nulo. fechahora (DATETIME): Fecha y hora en que se realizó la transacción. Este campo es obligatorio y no puede ser nulo. idmetodopago (INT): Llave foránea que referencia a la tabla metodospago. Indica qué método de pago se utilizó para la transacción.
