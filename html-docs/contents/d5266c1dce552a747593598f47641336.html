Description

Descripción: Esta tabla almacena información sobre los productos comprados a los proveedores, vinculando cada compra con su respectiva factura.

idfacturaproveedor (INT): Llave foránea que referencia a la tabla facturasproveedor. Indica a qué factura de proveedor corresponde la compra del producto. idproducto(INT): Llave foránea que referencia a la tabla productos. Indica qué producto se ha comprado. cantidad (INT): Cantidad del producto comprado. Este campo es obligatorio y no puede ser nulo. PRIMARY KEY (idfacturaproveedor, id_producto): Esta combinación de claves asegura que cada compra de un producto esté asociada con una factura específica y que no haya duplicados en la misma factura para un producto determinado.
