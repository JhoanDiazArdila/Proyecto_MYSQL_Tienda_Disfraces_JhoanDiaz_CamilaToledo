Description

Descripción: Esta tabla contiene información sobre los envíos realizados por la empresa, incluyendo detalles sobre el estado y la ubicación del envío.

idenvio (INT): Clave primaria, identificador único y autoincremental para cada envío. estado (ENUM): Estado del envío, que puede ser uno de los siguientes: 'Preparación', 'Recogido', 'En tránsito', 'Entregado'. Este campo es obligatorio y no puede ser nulo. direccionenvio (VARCHAR): Dirección a la que se envía la mercancía. Este campo es obligatorio y no puede ser nulo. idciudadpueblo (INT): Llave foránea que referencia a la tabla ciudadespueblos, indicando la ciudad o pueblo donde se realiza el envío. zipcode (INT): Código postal del destino del envío. Este campo es obligatorio y no puede ser nulo. fechaenvio (DATE): Fecha en que se realizó el envío. Este campo es obligatorio y no puede ser nulo. nittransportadora (BIGINT): Llave foránea que referencia a la tabla transportadoras, indicando qué transportadora se encarga del envío. idfactura (INT): Llave foránea que referencia a la tabla Facturas, indicando la factura que será enviada.
