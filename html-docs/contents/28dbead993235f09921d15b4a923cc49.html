Description

Descripción: Esta tabla contiene información sobre los clientes de la organización.

idcliente (INT): Clave primaria, identificador único y autoincremental para cada cliente. nombre (VARCHAR): Nombre del cliente. Este campo es obligatorio y no puede ser nulo. telefono (VARCHAR): Número de teléfono del cliente. Este campo es único y no puede ser nulo. correo (VARCHAR): Dirección de correo electrónico del cliente. Este campo es obligatorio y no puede ser nulo. direccion (VARCHAR): Dirección de residencia del cliente. Este campo es obligatorio y no puede ser nulo. idciudadpueblo (INT): Llave foránea que referencia a la tabla ciudadespueblos. Indica a qué ciudad o pueblo pertenece el cliente. zipcode (INT): Código postal de la dirección del cliente. Este campo es obligatorio.
