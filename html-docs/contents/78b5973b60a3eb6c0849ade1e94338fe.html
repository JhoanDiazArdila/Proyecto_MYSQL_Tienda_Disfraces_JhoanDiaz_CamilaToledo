ERDEntity
Proyecto_Base_datos :: Tienda_disfraces_Doña_Ruca :: facturas

Description

Descripción: Esta tabla almacena información sobre las facturas emitidas por la organización.

idfactura (INT): Clave primaria, identificador único y autoincremental para cada factura. fecha (DATE): Fecha en que se emitió la factura. Este campo es obligatorio y no puede ser nulo. total (DECIMAL): Total de la factura, representado como un valor decimal. Este campo es obligatorio y no puede ser nulo. idcliente (INT): Llave foránea que referencia a la tabla clientes. Indica a qué cliente se le emitió la factura. idtransacion (INT): Llave foránea que referencia a la tabla transacciones. Indica qué transacción corresponde a la factura. idempleado (INT): Llave foránea que referencia a la tabla empleados. Indica qué empleado emitió la factura.
