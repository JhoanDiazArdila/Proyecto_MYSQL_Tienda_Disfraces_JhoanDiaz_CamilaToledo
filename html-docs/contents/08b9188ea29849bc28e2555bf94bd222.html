Description

Descripción: Esta tabla almacena información sobre los gastos operativos incurridos por la empresa.

idgasto (INT): Clave primaria, identificador único y autoincremental para cada gasto operativo. idtipogasto (INT): Llave foránea que referencia a la tabla tiposgastos, indicando el tipo de gasto al que pertenece este registro. descripcion (TEXT): Descripción detallada del gasto. Este campo puede ser nulo. fecha_pago (DATE): Fecha en que se realizó el pago del gasto. Este campo es obligatorio y no puede ser nulo. valor (DECIMAL): Valor del gasto, representado como un número decimal. Este campo es obligatorio y no puede ser nulo.
