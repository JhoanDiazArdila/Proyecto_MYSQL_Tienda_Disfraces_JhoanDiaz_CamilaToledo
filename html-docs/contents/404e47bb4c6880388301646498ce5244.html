Description

Descripción: Esta tabla almacena información sobre las ventas de producto realizadas a través de las facturas emitidas.

idfactura (INT): Llave foránea que referencia a la tabla facturas. Indica a qué factura pertenece la venta del producto. idproducto(INT): Llave foránea que referencia a la tabla disfraces. Indica qué producto se ha vendido. cantidad (INT): Cantidad del producto vendido. Este campo es obligatorio y no puede ser nulo. PRIMARY KEY (idfactura, idproducto): Esta combinación de claves asegura que cada venta de un producto esté asociada con una factura específica y que no haya duplicados en la misma factura para un producto determinado.
