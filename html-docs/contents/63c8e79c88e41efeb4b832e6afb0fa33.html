Proyecto_Base_datos :: Tienda_disfraces_Doña_Ruca :: alquiler_disfraz

Description

Descripción: Esta tabla contiene información sobre el alquiler de disfraces, permitiendo un seguimiento detallado de las transacciones de alquiler.

idfactura (INT): Llave foránea que referencia a la tabla facturas. Indica a qué factura corresponde el alquiler del disfraz. idproducto (INT): Llave foránea que referencia a la tabla disfraces. Indica qué disfraz se ha alquilado. fechaalquiler (DATE): Fecha en que se realizó el alquiler del disfraz. Este campo es obligatorio y no puede ser nulo. fechadevolucion (DATE): Fecha en que se debe devolver el disfraz. Este campo es obligatorio y no puede ser nulo. valordeposito (DECIMAL): Monto del depósito requerido para el alquiler, representado como un valor decimal. Este campo es obligatorio y no puede ser nulo. cantidad (INT): Cantidad de disfraces alquilados. Este campo es obligatorio y no puede ser nulo. estadodeposito (ENUM): Estado del depósito, que puede ser "Pendiente", "Entregado" o "Cobrado". Este campo es obligatorio y no puede ser nulo. PRIMARY KEY (idfactura, iddisfraz): Esta combinación de claves asegura que cada alquiler de un disfraz esté asociado con una factura específica y que no haya duplicados en la misma factura para un disfraz determinado.
