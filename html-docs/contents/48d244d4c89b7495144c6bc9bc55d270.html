departamentos_estados

ERDEntity
Proyecto_Base_datos :: Tienda_disfraces_Doña_Ruca :: departamentos_estados

Description

Descripción: Esta tabla contiene información sobre los departamentos o estados dentro de un país. Cada departamento o estado está asociado a un país específico.

iddepartamentoestado (INT): Clave primaria, identificador único y autoincremental para cada departamento o estado. nombre (VARCHAR): Nombre del departamento o estado. Este campo es obligatorio y no puede ser nulo. id_pais (INT): Llave foránea que referencia a la tabla paises. Indica a qué país pertenece el departamento o estado.
