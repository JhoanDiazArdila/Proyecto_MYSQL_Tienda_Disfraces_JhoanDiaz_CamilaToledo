Proyecto_Base_datos :: Tienda_disfraces_Doña_Ruca :: accesorios

Description

Descripción: Esta tabla almacena los datos relacionados con los accesorios que se venden en la tienda, como tamaño, material, tipo de accesorio, color, y su relación con la tabla de productos.

idaccesorio (INT): Clave primaria, identificador único y autoincremental para cada accesorio. tamaño (VARCHAR): Define el tamaño del accesorio, como "pequeño", "mediano", "grande", o descripciones más específicas. idmaterial (INT): Llave foránea que referencia a la tabla materiales. Indica el tipo de material del que está hecho el accesorio, como plástico, cuero, metal, etc. idtipoaccesorio (INT): Llave foránea que referencia a la tabla tiposaccesorios. Especifica el tipo de accesorio, por ejemplo, joyería, sombreros, máscaras, etc. idproducto (INT): Llave foránea que referencia a la tabla productos. Cada accesorio es un tipo de producto, por lo que se vincula con una entrada en la tabla de productos. id_color (INT): Llave foránea que referencia a la tabla colores. Define el color predominante del accesorio. stock (INT): Cantidad de unidades disponibles en inventario para la venta. Relaciones:

idmaterial referencia a materiales(idmaterial): Conecta el accesorio con un material específico de la tabla de materiales. idtipoaccesorio referencia a tiposaccesorios(idtipoaccesorio): Vincula el accesorio con un tipo de accesorio predefinido. idproducto referencia a productos(idproducto): Relaciona el accesorio con una entrada general en la tabla de productos. idcolor referencia a colores(id_color): Asocia el accesorio con un color específico.
