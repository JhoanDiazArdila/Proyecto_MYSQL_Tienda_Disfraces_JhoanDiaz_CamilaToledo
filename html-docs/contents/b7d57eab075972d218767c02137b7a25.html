ciudades_pueblos

ERDEntity
Proyecto_Base_datos :: Tienda_disfraces_Doña_Ruca :: ciudades_pueblos

Description

Descripción: Esta tabla almacena información sobre las ciudades o pueblos que pertenecen a un departamento o estado específico.

idciudadpueblo (INT): Clave primaria, identificador único y autoincremental para cada ciudad o pueblo. nombre (VARCHAR): Nombre de la ciudad o pueblo. Este campo es obligatorio y no puede ser nulo. iddepartamentoestado (INT): Llave foránea que referencia a la tabla departamentos_estados. Indica a qué departamento o estado pertenece la ciudad o pueblo.
