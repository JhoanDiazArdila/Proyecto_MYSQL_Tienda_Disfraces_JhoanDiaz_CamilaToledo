Proyecto_Base_datos :: Tienda_disfraces_Doña_Ruca :: productos

Description

Descripción: Esta tabla almacena información sobre los productos generales que la tienda ofrece, como disfraces, maquillaje, decoraciones, etc. Cada producto tiene un identificador único y detalles relacionados con su nombre, descripción, precio de venta, costos, y a qué categoría y marca pertenecen.

idproducto (INT): Es la clave primaria, es un identificador único para cada producto, autoincremental. nombre (VARCHAR): Nombre del producto. Es un campo obligatorio. descripcion (TEXT): Descripción detallada del producto. precioventa (DECIMAL): El precio al cual se vende el producto. idcategoria (INT): Llave foránea que referencia la tabla categorias. Indica a qué categoría pertenece el producto (como disfraces, maquillaje, decoraciones, etc.). idmarca (INT): Llave foránea que referencia la tabla marcas. Especifica la marca del producto. costo (DECIMAL): El costo de adquisición del producto. Relaciones:

idcategoria referencia a categorias(idcategoria): Esto permite que cada producto esté asociado a una categoría específica (por ejemplo, cosméticos, disfraces, decoraciones, etc.). idmarca referencia a marcas(idmarca): Cada producto pertenece a una marca registrada en la tabla de marcas.
