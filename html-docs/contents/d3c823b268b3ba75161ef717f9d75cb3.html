Description

Descripción: Esta tabla almacena información sobre los empleados de la organización.

idempleado (INT): Clave primaria, identificador único y autoincremental para cada empleado. nombre (VARCHAR): Nombre del empleado. Este campo es obligatorio y no puede ser nulo. telefono (VARCHAR): Número de teléfono del empleado. Este campo es único y no puede ser nulo. correo (VARCHAR): Dirección de correo electrónico del empleado. Este campo es único y no puede ser nulo. direccion (VARCHAR): Dirección de residencia del empleado. Este campo es obligatorio y no puede ser nulo. idciudadpueblo (INT): Llave foránea que referencia a la tabla ciudadespueblos. Indica a qué ciudad o pueblo pertenece el empleado. zipcode (INT): Código postal de la dirección del empleado. Este campo es obligatorio. fechacontratacion (DATE): Fecha en la que el empleado fue contratado. Este campo es obligatorio. idpuestotrabajo (INT): Llave foránea que referencia a la tabla puestostrabajos. Indica el puesto de trabajo que ocupa el empleado. estado (ENUM): Estado del empleado, que puede ser "Activo" o "Inactivo".
