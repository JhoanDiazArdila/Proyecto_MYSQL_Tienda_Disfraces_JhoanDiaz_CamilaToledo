Description

Descripción: Esta tabla almacena información sobre los proveedores que suministran productos a la organización.

nit (BIGINT): Clave primaria, número de identificación tributaria del proveedor. Este campo es único y no puede ser nulo. nombreempresa (VARCHAR): Nombre de la empresa proveedora. Este campo es único y no puede ser nulo. nombrecontacto (VARCHAR): Nombre de la persona de contacto en la empresa proveedora. Este campo es obligatorio y no puede ser nulo. telefono (VARCHAR): Número de teléfono del proveedor. Este campo es único y no puede ser nulo. correo (VARCHAR): Dirección de correo electrónico del proveedor. Este campo es único y no puede ser nulo. direccion (VARCHAR): Dirección física del proveedor. Este campo es obligatorio y no puede ser nulo. idciudadpueblo (INT): Llave foránea que referencia a la tabla ciudades_pueblos. Indica la ciudad o pueblo donde se encuentra el proveedor. zipcode (INT): Código postal del proveedor. Este campo es obligatorio y no puede ser nulo.
