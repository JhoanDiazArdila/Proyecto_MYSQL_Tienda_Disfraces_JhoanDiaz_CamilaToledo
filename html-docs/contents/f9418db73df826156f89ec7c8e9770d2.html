Proyecto_Base_datos :: Tienda_disfraces_Doña_Ruca :: disfraces

Description

Descripción: Esta tabla contiene los detalles específicos de los disfraces que la tienda tiene en venta y alquiler. Almacena información sobre género, usuario al que está dirigido (niños o adultos), y la relación con el producto base, además del stock disponible tanto para venta como para alquiler.

iddisfraz (INT): Clave primaria, identificador único para cada disfraz, autoincremental. genero (ENUM): Especifica si el disfraz está diseñado para hombres, mujeres o si es unisex. tipousuario (ENUM): Define si el disfraz es para niños o para adultos. idtematica (INT): Llave foránea que referencia a la tabla tematicas. Define la temática del disfraz, como superhéroes, mitología griega, animales, etc. idcolor (INT): Llave foránea que referencia a la tabla colores. Indica el color predominante del disfraz. idtalla (INT): Llave foránea que referencia a la tabla tallas. Indica la talla del disfraz (por ejemplo, pequeña, mediana, grande). idproducto (INT): Llave foránea que referencia a la tabla productos. Cada disfraz está vinculado a un producto base en la tabla de productos. stockventa (INT): La cantidad de disfraces disponibles para la venta. stockalquiler (INT): La cantidad de disfraces disponibles para alquilar. precio_alquiler (DECIMAL): El precio al que se alquila el disfraz. Relaciones:

idtematica referencia a tematicas(idtematica): Esta relación asocia cada disfraz con una temática predefinida (superhéroes, anime, animales, etc.). idcolor referencia a colores(idcolor): Vincula cada disfraz con un color específico de la tabla de colores. idtalla referencia a tallas(idtalla): Asigna la talla del disfraz, que se gestiona en la tabla de tallas. idproducto referencia a productos(idproducto): Conecta el disfraz con su correspondiente entrada en la tabla de productos, para llevar un control general de todos los productos, ya sean disfraces, accesorios, maquillaje, etc.
